# Phase 24 Context: Governed Design System

Phase 23 decides narrative and semantic form. Phase 24 must resolve those decisions into governed design choices so a weak model never invents coordinates, typography, colors, decoration, or asset behavior.

The runtime truth is JSON. Layout geometry is normalized and owned by registries, not supplied by DeckPlan. Eight themes cover light, dark, technology, finance/investor, vibrant marketing, e-commerce editorial, education/training, and public-enterprise needs. Twenty-four page families each provide at least three deterministic variants. Every semantic form emitted by Phase 23 must resolve to a family and variant through explicit aliases and compatibility rules.

Phase 24 does not create PowerPoint shapes. It validates and resolves immutable theme, layout, component, brand, font, and asset specifications consumed by Phase 25.
