# Phase 28 Plan 01: Frozen Weak-Model Benchmark

## Objective

Build and execute a reproducible benchmark that distinguishes model ability from Skill support across fifteen business scenarios, three arms, two ordinary models, and repeated trials.

## Work Packages

- [ ] Add failing tests and strict schemas for scenarios, protocol, trials, hashes, deterministic scores, blind reviews, and aggregate verdicts.
- [ ] Freeze fifteen synthetic commercial briefs with fact IDs, prohibited inventions, narrative beats, expected archetypes/forms, capacity ranges, and asset conditions.
- [ ] Freeze three arms, two ordinary OpenCode model IDs, two repeats, deterministic seeds, prompt contracts, and release thresholds.
- [ ] Generate the complete 180-trial manifest with canonical IDs and hashes.
- [ ] Implement response ingestion that accepts real provider output only, records unavailable/invalid states, validates DeckPlans, and preserves raw evidence.
- [ ] Score fact retention, prohibited invention, compilation, density/rhythm, diversity, editability, QA hard gates, artifacts, and repeat agreement.
- [ ] Generate deterministic anonymized blind-review packets and validate completed human score sheets without exposing arm/model identity.
- [ ] Produce aggregate arm/model/scenario scorecards, before/after deltas, confidence/completeness evidence, and a threshold verdict.
- [ ] Execute available ordinary-model trials through OpenCode, without substituting synthetic outputs when provider calls fail.
- [ ] Generate governed fake-COM artifacts for available full-v5 trials and reserve native PPTX evidence for Phase 29.
- [ ] Run focused and complete suites, verify all hashes, independently audit conclusions, and document limitations.

## Exit

Phase 29 remains blocked if the scenario corpus or protocol can drift silently, model outputs are fabricated or post-edited, arms receive unequal source facts, trial hashes are incomplete, blind review leaks identity, score aggregation imputes missing trials, thresholds are weakened after results, or before/after claims lack recorded evidence.
