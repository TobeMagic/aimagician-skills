# Platform Query Playbook

Use this when anonymous bulk retrieval is not enough.

## Priority order

### 1. OpenAlex

Use for broad discovery and theme coverage.

### 2. arXiv

Use for recent preprints and trend tracking.

### 3. Crossref

Use for DOI and metadata expansion.

### 4. CVF Open Access

Use for vision, 3D, scene memory, active perception, and related conference supplementation.

### 5. Semantic Scholar

Use when citation-rich expansion or similarity search matters and the environment supports credentials.

## Manual follow-up layers

When the automated stack is not enough, supplement manually with:

- ACL Anthology
- OpenReview
- IEEE Xplore
- Springer / ScienceDirect

But do not pretend these are covered by the default pipeline unless they were actually fetched.

## When to supplement manually

Only do manual or venue-specific supplementation when it helps answer a concrete question, such as:

- whether a direction is novel enough
- whether a benchmark claim is honest
- whether a venue narrative is plausible
- whether a module already exists in a very close form
