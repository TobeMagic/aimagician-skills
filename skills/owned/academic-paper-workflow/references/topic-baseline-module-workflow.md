# Topic, Baseline, and Module Workflow

This is a compatibility bridge for older routes. For new work:

- use `idea-selection-workflow.md` for topic and idea scoring
- use `method-synthesis-workflow.md` for baseline/module synthesis
- use `contribution-evidence-workflow.md` for contribution framing

## Minimal Route

1. Score candidate ideas for field fit, novelty room, baseline/data availability, evidence cost, and venue fit.
2. Select one reproducible baseline and one backup baseline.
3. Build a module bank tied to baseline limitations.
4. Choose a synthesis route: baseline+module, method transfer, application adaptation, benchmark/resource, or review/survey.
5. Map every claim to evidence before drafting.

## Output

Return:

- idea scorecard
- baseline shortlist
- module bank
- method synthesis route
- ablation matrix
- contribution boundary
